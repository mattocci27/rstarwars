rstarwars

StarWars on R console

Installation

devtools::install_github("mattocci27/rstarwars")

Usage

starwars()

Specify episode number

starwars(episode = "sw1")

Specify frame rate

starwars(episode = "sw1", fps = 20)

Episodes

  • SW1
  • SW2
